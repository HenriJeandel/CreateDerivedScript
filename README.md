# CreateDerivedScript
A Unity package allowing to create a script with a class deriving from the selected script.

## Install
Add this package using "install package from git URL"

## Use 
When selecting a MonoScript (a C# script in Unity), right-click, then Create > Scripting > C# Script (Inherit Selected)
Rename the newly created script with the name of the class you want.

## Unity Version
Last tested on Unity 6000.0.59f2.
